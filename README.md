# Mini Cloud Drive (Compute & Elasticity · Cloud Computing Project)

A mini Google Drive–style web app built with **FastAPI** and **AWS S3**.

## Features

- User signup / login (cookie-based sessions)
- Upload, list, download, and delete files
- Files stored in **AWS S3**, metadata in SQLite
- Clean, Google Drive–inspired UI

## Tech Stack

- **Backend:** FastAPI, SQLAlchemy, Jinja2 templates
- **Storage:** AWS S3
- **Database:** SQLite (local)
- **Frontend:** HTML + CSS (custom responsive design)
- **Language:** Python 3.11+
